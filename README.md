# wlhangul

A Hangul input method for Wayland.

> [!IMPORTANT]
> This project has [moved to gitlab.freedesktop.org].

## Building

Depends on libhangul, Wayland and libxkbcommon.

    meson build/
    ninja -C build/
    build/wlhangul
